# Code of Conduct

## Our Pledge

We are committed to fostering an open, welcoming, inclusive, and respectful community for everyone, regardless of age, experience, background, identity, or skill level.

We pledge to make participation in this project a positive experience for all contributors.

---

## Our Standards

Examples of behavior that contributes to a positive environment include:

* Being respectful and considerate.
* Accepting constructive feedback gracefully.
* Collaborating openly.
* Helping newcomers.
* Using welcoming and inclusive language.
* Respecting different viewpoints and experiences.

Examples of unacceptable behavior include:

* Harassment or discrimination.
* Personal attacks or insults.
* Trolling or intentionally disruptive behavior.
* Hate speech.
* Threats or intimidation.
* Publishing private information without permission.
* Any unlawful or unethical conduct.

---

## Responsibilities

Project maintainers are responsible for:

* Clarifying acceptable behavior.
* Reviewing reported incidents fairly.
* Taking appropriate corrective action when necessary.
* Maintaining a respectful environment.

---

## Reporting Issues

If you experience or witness unacceptable behavior, please report it privately to the project maintainers.

Include:

* A description of the incident.
* Date and time.
* Any relevant screenshots or logs.
* Any additional context that may assist the investigation.

Reports will be handled confidentially whenever possible.

---

## Enforcement

Project maintainers may take appropriate action, including:

* Issuing warnings.
* Removing inappropriate content.
* Temporarily restricting participation.
* Permanently banning users who repeatedly violate this Code of Conduct.

---

## Scope

This Code of Conduct applies to:

* GitHub Issues
* Pull Requests
* Discussions
* Project documentation
* Community forums
* Any official communication channels related to this project
